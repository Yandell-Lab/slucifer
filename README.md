# Slucifer
BLAST wrapper to generate higher sensitivity but lower specificity results (off-road BLAST)

## Quick install
```
git clone https://github.com/Yandell-Lab/slucifer.git
cd slucifer/src
perl Build.PL
./Build installdeps
./Build installexes
./Build install
```
